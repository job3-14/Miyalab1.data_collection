    class Crawler(builtins.object)
     |  Crawler(args, domains, sleep_time=3)
     |  
     |  カテゴリごとに記事をスクレイピングするためのクラス
     |  
     |  Methods defined here:
     |  
     |  __init__(self, args, domains, sleep_time=3)
     |      初期化します
     |  
     |  crawl_article_page(self, category_name, urls, output_path)
     |      指定された回数だけ記事をサーバーから取得し、jsonファイルとして書き込みを行う
     |  
     |  crawl_top_page(self, start_urls, url_pattern)
     |      start_urlsから記事のurlセットを作成し、返す。
     |      もし見つからなかった場合は空のセットを返す。
     |  
     |  define_format(self, article_id, category_name, url, article)
     |      id,category,url,title,bodyを辞書型にして返す
     |  
     |  extract_title_and_body(self, html_document)
     |      入力された記事htmlからタイトルと本文を返す
     |  
     |  get_html_document(self, url)
     |      urlからテキストをダウンロードし、テキスト本文(html)を返す
     |  
     |  run(self, category_name, start_urls, url_pattern)
     |      スクレイピングを実行します
     |  
     |  ----------------------------------------------------------------------
     |  Static methods defined here:
     |  
     |  extract_urls(url_pattern, html_document)
     |      BeatifulSoupを使用して解析する。引数のdocumentからurlリストのみを抽出し返す。
     |  
     |  get_article_id(url)
     |      urlから最後の/の後の記事idのみをreturnする。
     |  
     |  join_path(*a_tuple)
     |      ファイルを保存するディレクトリを返す。
     |      引数(*a_tuple)は複数の引数をタプルとして受け取る
     |  
     |  make_directories(path)
     |      出力するディレクトリを作成する
     |  
     |  write_json_file(file_name, a_dict)
     |      入力された辞書をjsonファイルとして保存
     |  
     |  ----------------------------------------------------------------------
     |  Data descriptors defined here:
     |  
     |  __dict__
     |      dictionary for instance variables (if defined)
     |  
     |  __weakref__
     |      list of weak references to the object (if defined)
